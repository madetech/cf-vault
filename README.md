# Cloud Foundry Vault

`cf-vault` is the tool you use to syncronise your application secrets stored in [Vault][Hashicorp's Vault] to [CloudFoundry][Cloud Foundry] providers like [Pivotal][Pivotal].

```
rake cf:vault
```
